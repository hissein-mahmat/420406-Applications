Activité d'accueil

Mise en page

1 point la mise en page respecte l'image suivante (cliquez pour élargir)

Boutons

1 point les boutons fonctionnement. Scanner doit démarrer l'application de lecture de codes QR. Ajouter doit amener à l'activité de création d'une question.

Code QR

2 points Si le code QR représente une question valide (voir critères TP2), une nouvelle question est créée en base de données.

Ce TP nécessite que les données soient sauvegardée dans la BD SQLite sur l'appareil Android. Il est recommandé de modifier le service produit dans le TP2 pour qu'il utilise un DAO Room.

Les points de BD ne seront accordés que si les données sont enregistrées dans SQLite.

Création d'une question

Mise en page

1 point la mise en page respecte l'image suivante (cliquez pour élargir)

Fonctionnement correct

2 points Si on entre une question valide (voir critères TP2), une nouvelle question est créée en base de données.

Erreur

1 point Il y a un message d'erreur significatif en cas d'erreur.

ATTENTION! SI ON NE PEUT PAS CRÉER DE QUESTION, ON NE PEUT PAS CORRIGER LA SUITE.

Liste des questions

Mise en page

1 point la mise en page de chaque élément de la liste respecte l'image suivante (cliquez pour élargir)

Tri

1 point La liste doit être triée selon les critères indiqués dans le TP2.

Comportement

2 points Un appui sur le bouton avec l'icône amène aux résultats. Un appui sur tout autre endroit de la ligne amène à l'écran de vote.

Écran de vote

1 point La mise en page doit correspondre avec un moyen d'enregistrer une note entre 0 et 5 inclus et un champ pour entrer le nom du votant.

2 points Un vote est créé en base de données avec la note et le votant.

1 point Il y a un message d'erreur significatif si le votant a déjà voté pour cette question.

ATTENTION! SI ON NE PEUT PAS CRÉER DE VOTE, ON NE PEUT PAS CORRIGER LA SUITE.

Écran de résultats

2 point La mise en page correspond et permet d'afficher la moyenne, l'écart type et la distribution des votes par note.

1 point Un message s'affiche s'il n'y a aucun vote pour cette question à ce moment.

Menu de l'activité

  • 1 point Supprime les votes doit supprimer tous les objets votes dans la base de données
  • 1 point Supprime les questions doit supprimer toutes les questions sauf celles où il y a des votes
  • 1 point La liste se met à jour après suppression (éléments et ordre)

Base de données et DAO

Les fonctionnalités demandées dans ce travail nécessite des opérations au niveau de la base de données et sans doute de nouvelles méthodes au niveau de votre service (pour effectuer des validations, tris et autres autour des accès à la BD).

4 points test Tests s'exécutant sur l'appareil Android pour valider les différentes méthodes.

Tu peux t'aider des tests de ton TP2.

Fonctionnement global

5 points négatifs Fonctionnement correct et sans plantage. Ce pointage fonctionne en négatif. Si l'application fonctionne correctement en tout temps, vous conservez votre note. Dans le cas contraire, vous perdez des points avec un maximum de 5.

  • plantage de l'application -1 point
  • interface illisible -1 point
  • information affichée incohérente -1 point
  • autre cas ...
