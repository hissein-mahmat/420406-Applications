Exercices de la semaine

Exercice StockageDernierArret

Dans un nouveau projet, vous devez programmez la persistance suivante.

  • Une seule activité
  • À la fermeture de l'activité (événement onDestroy ici), l'activité sauvegarde sur l'appareil l'heure (vous choisissez le format et le type de sauvegarde SharedPref/fichier/SQLite)
  • Au démarrage de l'activité, on affiche dans un TextView l'heure du dernier arrêt de l'application. Au premier démarrage de l'application, on n'affiche rien mais on fait apparaître un Toast indiquant un tout premier démarrage.

Exercice Persist

En partant du projet Persist fourni sur le repository github, vous devez:

  • Complétez le code pour la partie SQL dans l'interface graphique (qui ne marche pas)
  • Arrangez le code pour que le ListView affiche les éléments persistés au démarrage (et pas uniquement après un ajout ou un vidage)
  • Ajoutez une traduction en anglais à l'interface graphique fournie
