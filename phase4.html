Taux de taxe

1 point code 1 point test Implante 2 taux de taxes: un taux de taxe à 5% pour des produits essentiels comme le lait, le pain etc. Un autre taux de taxe à 35% pour les autres produits comme la pizza surgelé ou les sodas.

Inclus des produits pour chaque taux de taxe dans ton inventaire standard (les 9 produits).

Rabais

Pour chaque type de rabais, tu dois :
  • programmer l'impact sur le ticket de caisse final
  • produire des tests unitaires pour valider le bon fonctionnement
  • ajouter les éléments d'interface graphique pour configurer le rabais
  • inplanter la persistance de chaque type de rabais

2 pour 1

1 point code 1 point interface graphique 1 point test Chaque produit peut être en 2 pour 1. Exemple : un client achète 7 bouteilles d’eau en 2 pour 1, on peut donc appliquer 3 fois le rabais: 

(1 + 1) + (1 + 1) + (1 + 1) + 1

Le client doit donc payer le prix de 4 bouteilles.

Pas de taxes si on dépasse 100$

1 point code 1 point interface graphique 1 point test On doit pouvoir activer ce rabais qui annule les taxes quand le montant hors taxe dépasse 100$ pour un achat.

Calcul des résultats

1 point code 2 points test Tu dois toujours appliquer les rabais de la manière la plus profitable au client. Par exemple, un client peut acheter pour 95$ de produits plus 2 produits à 4 dollars en 2 pour 1.

  • si j'applique le 2 pour 1, le client paie 99$ plus les taxes soit 113.85$
  • si j'oublie le 2 pour 1, le client paie 103$ puisqu'on ne charge pas les taxes

Validation

4 points négatifs Tous les endroits où un utilisateur entre de l'information doivent être validés. Tu perds des points pour chaque endroit où j'arrive à entrer de l'information invalide. Ceci inclut le code barre ... 

public class BarcodeValidator {

    /**
     * Voir  http://en.wikipedia.org/wiki/Universal_Product_Code#Check_digits
     * @param s
     * @return
     */
    public static boolean isUPCValid(String s){
        if (s.length() != 12) return false;     // doit faire 12 de long
        if (!s.matches("[0-9]*")) return false; // doit uniquement contenir des chiffres
        // somme des positions impaires (1er, 3e, 5e, etc.)
        int sumOdd = 0 ;
        // pareil pour les positions paires (2e, 4e, 6e etc.)
        int sumEven = 0;
        char[] array = s.toCharArray();
        for (int i = 0 ; i < array.length - 1 ; i++){
            int digit = Integer.parseInt(array[i]+"");
            if (i%2==0) sumOdd += digit;
            else sumEven += digit;
        }
        // 3 * somme des impairs + somme des pairs ... ensuite modulo 10
        int result = (3*sumOdd + sumEven)%10;
        // If the result is not zero, subtract the result from ten.
        result = result!=0?10-result:0;
        // on compare avec le dernier chiffre
        int lastDigit = Integer.parseInt(array[array.length-1]+"");
        return result == lastDigit;
    }
}
