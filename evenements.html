Exercices de la semaine

Exercice EvenementsMultiple

Programme une application avec un bouton qui lance un évémements via un post sur un bus. Programme un Subscribe dans l'activité de l'application qui affiche un Log.

Après avoir passé plusieurs fois la tablette du mode paysage au mode portrait puis au mode paysage. Appuis sur le bouton et observe le Log. Y a-t-il plusieurs Log pour chaque événements? Explique.

Exercice EvenementsHéritage

Programme une application avec 3 boutons qui lance respectivement un événement de type:

  • classe A qui contient un champ de type date correspondant à la date d'appui sur le bouton
  • classe A1 une sous-classe de A
  • classe A2 une autre sous-classe de A

Écris 3 subscribe affichant des messages différents selon le type d'événements dans le Log. En déclenchant les différents événements (en appuyant sur les boutons), observe le Log et explique.
